# Town Media AI - AI-Powered Digital Marketing Agency

A modern, responsive website for Town Media AI - an AI-powered digital marketing agency that automates and optimizes paid ads and social media growth across all major platforms.

## Features

- **AI-Powered Marketing**: Advanced AI optimization for campaigns across Facebook, Instagram, Google, TikTok, YouTube, and Snapchat
- **Organic Growth Tools**: Real follower growth for social media platforms
- **Smart Analytics Dashboard**: Real-time performance metrics and insights
- **AI Assistant**: 24/7 chatbot for marketing recommendations and order tracking
- **Multilingual Support**: Full Arabic and English interface support
- **Responsive Design**: Optimized for all device sizes
- **Modern UI**: Glass morphism effects, gradients, and smooth animations

## Tech Stack

- **Framework**: Next.js 13 with App Router
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Animations**: CSS transitions and transforms
- **Deployment**: Static export ready

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

```
├── app/
│   ├── globals.css          # Global styles and Tailwind config
│   ├── layout.tsx           # Root layout with metadata
│   └── page.tsx             # Main homepage
├── components/
│   ├── ui/                  # shadcn/ui components
│   ├── layout/              # Navigation and Footer components
│   └── sections/            # Page sections (Hero, Features, etc.)
├── lib/
│   └── utils.ts             # Utility functions
└── public/                  # Static assets
```

## Key Sections

- **Hero Section**: Main value proposition with animated stats
- **Features Section**: AI-powered marketing capabilities
- **Platforms Section**: Supported social media and ad platforms
- **Analytics Section**: Dashboard preview with performance metrics
- **Pricing Section**: Three-tier pricing plans (Free, Premium, Enterprise)
- **Testimonials**: Customer success stories
- **AI Assistant**: Chatbot interface demonstration

## Customization

The website is fully customizable with:
- Color schemes via CSS variables
- Component-based architecture for easy modifications
- Responsive breakpoints for all screen sizes
- Modular sections that can be reordered or modified

## Deployment

Build for production:
```bash
npm run build
```

The project is configured for static export and can be deployed to any static hosting service.
